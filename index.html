Comparing GCC and RNFL Thickness for Glaucoma Detection with XGBOOST

Tools: Python, and Machine Learning

Compared GCC and RNFL thickness to predict pattern standard deviation using the XGBOOST regressor and evaluated the model with RMSE. We then expanded the feature set to include age, sex, CDR, and IOP measurements for Glaucoma prediction using an XGBOOST classifier. This approach yielded an impressive 85% accuracy and an AUC score of 0.84
